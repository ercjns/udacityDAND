Project based largely on the sample project provided by Udacity: https://gist.github.com/carlward/54ec1c91b62a5f911c42#file-sample_project-md

ensmallen.py
=====
Script provided by Udacity to create a sample of nodes in an osm xml file that is smaller and therefore faster to work with while debugging.

schema.py
=====
SQL Schema in python format provided by udacity: https://gist.github.com/swwelch/f1144229848b407e0a5d13fcb7fbbd6f

load.py
=====
Processes a OSM XML file and creates csv files with the Udacity provided schema. Code adapted from the case study lesson. 
Once csv files are generated, they are loaded into a SQL database.
Includes a few functions which clean the data before it is inserted in the database.

denver_sample.osm
=====
9MB Sample of the dataset generated by using ensmallen.py with n=100.
